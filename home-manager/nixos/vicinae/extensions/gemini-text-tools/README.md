# Gemini Text Tools

Small Vicinae extension for Raycast-AI-style text rewriting with Gemini BYOK.

## Commands

- Rewrite Text
- Custom Rewrite
- Improve Writing
- Fix Spelling and Grammar
- Make Shorter
- Make Longer
- Professional Tone
- Friendly Tone

## Preferences

Configure the extension in Vicinae Preferences > Extensions:

- Gemini API Key
- Model
- Preferred Source
- Default Quick Action
- Additional Instructions
